# PythonToUnityOSC
OpenSound Control between Python and Unity

# Usage
- Start `OSCServer` scene in Unity.
- Move to `Python` directory in PC console. 
- Type ```python client.py``` in PC console.
- Messages will be shown in Unity Editor console.
